# Mnemosyne

Aplicação web desenvolvida utilizando JavaScript, com foco em praticar desenvolvimento frontend e interação com o usuário.

## 🚀 Funcionalidades
- Interação dinâmica com o usuário
- Estrutura básica de interface

## 🛠️ Tecnologias
- JavaScript
- HTML
- CSS

## 📚 Objetivo
Este projeto foi criado com o objetivo de praticar conceitos de frontend e aprimorar habilidades de desenvolvimento.
